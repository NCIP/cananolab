Protocol Search Results
If you click the Search button having defined no parameters, all of the protocols located in the search display in the search results.
Once you submit a search for a protocol (Searching Existing Protocols), search results display in the caNanoLab screen. Results show pertinent protocol information and a URI to each report. Columns are sortable by page by clicking on the column heading.
Click the Protocol Name URI to locate and open a protocol file, displayed as a Protocol Update Page.
