Adding Composition Files
This section of the [Sample] Composition category allows you to add files relevant to the sample independently of the entity or chemical associations that you have defined. You can add as many files as you wish.
To add a file, follow these steps.
Click [Other] to enter an alternative to the selections listed. At that point, your entry is enclosed in [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear.
1.
After you have clicked the Composition button on the Navigation Tree, click the Add button corresponding to Composition File on the All tab or select the Composition File tab itself where you can also click the Add button.
2.
Select the Upload button, then click the Browse button to navigate to the report or publication to be associated with the sample.
– OR –
Click the Enter File URL button and enter the URL where the document is located.
3.
For the file to be added, specify or enter the following:
a.
File Type (required): Document, Graph, Image, Spreadsheet
b.
File Title (required)
c.
Keywords to associate with the file
d.
Description: Enter any additional information for the file that the form does not already include.
4.
In the Visibility field, select the group(s) to be granted access to the selected file.
5.
When you have finished, click Submit to add the referenced file(s) to the sample or click Reset to clear all fields on the form.
After adding one or more files, continue the process described in Adding Nanomaterial Entity Composition Annotations.
You can add as many files as you wish.
For more information about working with publications, see Publications Overview.
