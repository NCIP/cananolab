Understanding the Navigation Tree
Once you submit or search and select an existing nanoparticle with the Nanoparticles menu option (see Nanoparticles Overview), the Navigation Tree heading displays in the left-hand panel of caNanoLab with the selected nanoparticle name. The Navigation Tree comprises categories which you can use to add annotations to the nanoparticle. Each category is a button, and when you click a button, options may expand below to show subcategories. Click the heading again, and the subcategories collapse.
The following list provides a general description of how you can use each category.
•
General Information: Display general information about the nanoparticle.
•
Function: Add actions that the nanoparticle can perform or purposes for which the nanoparticle is used. For more information, see Function Category.
•
Composition: Add composition properties to the nanoparticle. For more information, see Composition Category.
•
Physical Characterization: Add distinctive characteristics or essential features of the nanoparticle determined through analytical methods. You can also add physical characteristics to determine the material and structural properties of a nanoparticle. For more information, see Physical Characterization Category.
•
Invitro Characterization: Add distinctive characteristics or essential features of the nanoparticle determined through analytical methods. You can also annotate nanoparticles with laboratory results obtained under in vitro conditions. For more information, see In Vitro Characterization Category.
•
Associated Files: Currently not available.
•
Reports: Lists report files with characterization project results for the selected nanopartical. These files are added using the Reports menu option.
•
In Vivo Characterizations: Currently not available.
If information is filled out for the nanoparticle, you can add to or edit items with the Add or Delete buttons. If you do not have certain security privileges, the Add and Delete buttons will be hidden, and you will have read-only access to the items.
