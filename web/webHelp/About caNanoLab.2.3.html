About caNanoLab
Welcome to the cancer Nanotechnology Laboratory (caNanoLab) portal. caNanoLab is a data sharing portal designed for laboratories performing nanoparticle assays. caNanoLab provides support for the annotation of nanoparticles with characterizations resulting from physical and in vitro nanparticle assays, and the sharing of these characterizations in a sucure fashion.
caNanoLab provides support and functionality for following features:
•
Protocol management
•
Nanoparticle submission and search
•
Nanoparticle annotation creation, deletion, and search
•
Report management
•
Role-based security
caNanoLab source code is available at the National Cancer Institute Center for Bioinformatics (NCICB) download center: http://ncicb.nci.nih.gov/download/.
For more information about these features and their related tasks, see caNanoLab Functionality Overview.
