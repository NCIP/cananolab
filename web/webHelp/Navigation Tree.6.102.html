Displaying a Summary View of In Vitro Oxidative Stress Characterizations
If there are multiple Oxidative Stress In Vitro Characterizations added in the Navigation Tree, you can display a summary view of all the characterizations in one table. With additional icons, you can print the table or save the table to a spreadsheet file. You can also display and print the individual detail views of the category’s characterizations in one table. To display all the characterizations in one table, follow these steps:
1.
Click the Oxidative Stress subcategory. The appropriate In Vitro Characterization - Oxidative Stress page opens.
2.
The page opens listing all the characterizations in one table, one characterization per row.
3.
The following table lists and describes the fields that appear for the characterization.
If the table is missing information or empty, the following fields were not filled in when the characterization was added
Field
Description
View Title
A unique and meaningful name for the characterization.
Characterization File
A link to or a graphic representation of each characterization file which may be a graph, spreadsheet or image, or derived data. The characterization files are numbered in the order they were defined.
To view the characterization file graphic, click the graphic in the table. The graphic opens in a new, larger window.
Instrument Type
The instrument used to obtain data, the abbreviation of the instrument, and the manufacturer of the instrument.
4.
The following table lists and describes the icons and additional tasks that you can perform:

Print a detailed summary of the individual characterizations in the category.
Save the summary table to a spreadsheet file.
