Searching Existing Nanoparticles
You can access this page after logging into caNanoLab or you can access publicly available nanoparticle information from the Login page.
In the conventional sense, the term “sample” can refer to either of two entities in caNanoLab. This topic describes performing a search for a “Nanoparticle” sample type, that is, a nanoparticle sample to which annotations have been added. For more information, see Samples Overview.
A Nanoparticle search looks for a nanoparticle based on annotations that have been added or characterization or data files that have been uploaded. This search feature is accessible even to users for whom other search features are restricted. To launch a search for a specific nanoparticle, follow these steps:
1.
Click the Nanoparticles caNanoLab menu option. On the Manage Nanoparticle Home page, click the Search Existing Nanoparticles link. This opens the Search Nanoparticles form.
At the top of the page is the Grid Search link that enables you to search for nanoparticles on remote hosts. For more information, see Performing a Remote Nanoparticle Search.
2.
Enter text or select options as appropriate (described in the following table). In any text field, you can add an asterisk for a wild-card search.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search.

Characterization Type
Click from the displayed list on the left, the characterization type hypertext link that applies to the nanoparticle to be searched.
When you click a hypertext link, Characterization sub-categories display in the right-hand pane of the row. Select one or more of the items in the list. (CTRL + shift to select more than one.)
Summary/Description
Enter one word per line that summarizes or describes the characterization or uploaded file. Select the appropriate button for the search target: Characterization or Characterization file/Derived Data.
Keywords
Enter keywords, one word per line, that can be used for the nanoparticle or characterization search. These would correspond to keywords entered on the Submitting a Nanoparticle - General Information form when annotating a nanoparticle. Select the appropriate category for the search, nanoparticle or characterization..
If you click the Search button having defined no parameters, all of the caNanoLab nanoparticles will be listed in the Search Results
To clear all fields and start again, click the Reset button.
3.
To execute the search, click the Search button.
