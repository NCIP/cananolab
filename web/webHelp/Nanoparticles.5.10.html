Remote Nanoparticle Search Results
If you click the Search button having defined no parameters, all of the nanoparticles located in the remote search display in the search results.
Once you have submitted a search for Remote Nanoparticle Report (Performing a Remote Nanoparticle Search), search results display in table format in the caNanoLab screen. They are described in the following table. Columns are sortable by clicking the column heading.
 
Only public data can be retrieved from a remote search.
Remote Nanoparticle Search Results Columns
Description
Particle ID
The fully qualified name of the nanoparticle search hit.
Particle Source
The name of the source of the original nanoparticle. For instructions for each type, see Composition Category.
Particle Classification
The name of the nanoparticle classification. Example: Organic:Hydrocarbon.
Particle Type
The name of the nanoparticle category to which the nanoparticle belongs. Example: Dendrimer
Particle Functions
A list displaying functions that have been entered for the corresponding Particle ID.
Particle Characterizations
A list displaying characterization annotations that have been entered for the corresponding Particle ID.
Grid Node Host
Remote node(s) queried.
To view particle data, click the Particle ID hypertext link. Particle details are in read-only format.
