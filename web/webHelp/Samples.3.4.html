Managing Samples
The Sample Links on the Manage Sample/Aliquot Home page provide the following options:
•
Creating a New Sample: This takes you to a form to enter information for a new sample you want to submit. For more information, see Creating a New Sample.
•
Search Existing Samples: This takes you to a form for launching a search for an existing sample. For more information, see Searching Existing Samples.
