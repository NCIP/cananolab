Reports Overview
Reports search looks for submitted reports associated with multiple nanoparticles. This feature is accessible even to users for whom access to nanoparticles in caNanoLab is restricted. When you click the Reports option on the caNanoLab menu bar, the Manage Nanoparticle Home page appears. The Nanoparticle Links panel on the Manage Nanoparticle Home page provides the following options:
•
Create a New Report: This takes you to a form for creating a new caNanoLab report. For more information, see Submitting a Nanoparticle Report.
•
Search Existing Reports: This takes you to a form to launch a search for an existing nanoparticle report. For more information, see Searching Nanoparticle Reports.
For more information about nanoparticles, see Description of a Nanoparticle.
