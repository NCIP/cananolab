Adding an In Vitro Characterization
When you click the Navigation Tree In Vitro Characterization button, subcategories expand under the heading. (Click the button again to collapse the subcategories). You can annotate nanoparticles with laboratory results obtained under in vitro conditions. If annotation information is filled out for the nanoparticle, the annotations are listed under the subcategories.
Click the Add button next to the in vitro characterization category you want to add or modify. The page that opens is the same for all characterizations, except for slight customizations that exist and are noted.
While you may have the option to add or delete function details with the Add and Delete button, if you do not have certain security privileges, the Add and Delete buttons are hidden, and you have read-only access to the existing items.
Note that all in vitro characterizations are listed under the Toxicity category. Each page is described in online help in its own unique topic. For information and links specific to each, see Toxicity Category.
