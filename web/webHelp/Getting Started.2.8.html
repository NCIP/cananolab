Registering a New User
New users of caNanoLab are registered using the User Provisioning Tool (UPT), administered by an administrator. The administrator assigns all users to group “Public” when their user accounts are first created within the UPT tool. For more information, see Configuring a New User.
See Permissions and Visibility for more information.
