The Navigation Tree > Function Category
Function Category
When you click the Navigation Tree Function button, four subcategories expand under the heading (Click the button again to collapse the subcategories). You can use these subcategories to add the actions the nanoparticle can perform or the purposes for which the nanoparticle is used. The subcategories include the following:
Diagnostic Imaging
Diagnostic Reporting
Targeting
Therapeutic
If annotation information is filled out for the nanoparticle, the hyperlinked names of the annotations are listed under each subcategory. To edit the annotation, click the hyperlink. You may have the option to add or delete function details with the Add and Delete button.
If you do not have certain security privileges, the Add and Delete buttons will be hidden, and you will have read-only access to the existing items.
