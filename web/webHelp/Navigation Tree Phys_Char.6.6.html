Adding a Physical State Physical Characterization
In caNanoLab Nanoparticle Characterization is defined with Physical and In Vitro Characterizations.
Physical State is a Physical Characterization.
To record physical state data for a particle, follow these steps.
1.
On the Navigation Tree, click Physical Characterizations.
2.
Next to the Physical State heading, click the Add button.
If you do not have certain security privileges, the Add buttons will be hidden, and you will have read-only access to the existing items.
3.
In the Physical Characterization-Physical State page that opens, complete the following fields. (Asterisks indicate required fields.)
Some fields may contain the Other option. Click Other to enter an alternative to the selections listed. At that point, your entry is enclosed in [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear.

Characterization Source*
Select or enter the source from which the characterization data is derived, such as a vendor or a laboratory (NCL).
View Title*
Enter a unique and meaningful name for the characterization. The title you enter will display on the Navigation Tree. The asterisk indicates a required field.
Characterization Date
Select from the calendar or enter the date the characterization was made. Acceptable format: dd/mm/yyyy
Protocol Name - Version
Select from the drop-down list the protocol from which the data is derived. A hyperlink of the protocol text file may appear. Click the hyperlink to open or save the file.

Copy to Other Nanoparticle Samples
If appropriate, this option copies files and summary information to one or more selected particles from the same vendor. Select one or more particle(s) to which the copied data is to be transferred. (Only appropriate particles are listed.) For more information, see Copying Characterizations.
Also Copy Derived Data?
Select this button to copy derived data, as well, to the selected particle.
4.
When you have finished, click Submit or click Reset to clear all fields on the form.
In the Navigation Tree, a hypertext link for the new characterization is added under Physical State.
