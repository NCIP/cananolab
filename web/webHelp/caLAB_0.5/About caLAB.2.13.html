Viewing an Aliquot
Once an aliquot has been created and added to an Assay Run, you can view aliquot details. To do so, follow these steps:
1.
In the workflow tree, select a Run that you want to examine.
2.
Click the (+) symbol in front of the In folder (an aliquot is always part of the input).
If Aliquots have been added to the Run, they are listed in the In folder.
3.
Click on the Aliquot name to open the details page for the aliquot.
From this page, you can mask the aliquot.
