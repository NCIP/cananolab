Sample Container Search Results
Sample Container Search Results display in the viewer window in table format. Column headings are appropriate for the container type for which you searched, samples or ali­quots. For any hit, click on a hypertext link in the Action column (last) to open detailed information about the located components.
Sample Search Hits
Click View Details to open details about the sample. Click Show Aliquots to display details about aliquots created from the sample hit.
Aliquot Search Hits
Click View Details to open details for aliquots created from the sample. Details for only one aliquot display at a time; it includes information about its source sample.
For more information about configuring a sample search, see Searching Sample Con­tainers.
