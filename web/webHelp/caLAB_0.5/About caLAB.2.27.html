Searching Sample Containers
To launch a search for a specific sample or aliquot, follow these steps:
1.
Select Search > Search Sample. This opens the Search Sample Container page.
You can search for two container types: Sample Container or Aliquot Con足tainer.
2.
Enter text or select options as appropriate (described in the following table). In any text field, you can add an asterisk for a wild-card search. For example, in searching for sample NCL-8-234, you could enter NCL-8*.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search. If you click the Search button having defined no parameters, all of the caLAB sample or aliquot containers will be listed in the Search Results.
