Adding a Platelet Aggregation In Vitro Characterization
In caNanoLab Nanoparticle Characterization is defined with Physical and In Vitro Characterizations.
Platelet Aggregation is an In Vitro Characterization.
In the Navigation Tree the Add button next to the Platelet Aggregation name enables you to add in vitro characterizations to caNanoLab.
If you do not have certain security privileges, the Add buttons will be hidden, and you will have read-only access to the existing items.
To record plate aggregation data for a particle, follow these steps:
1.
On the Navigation Tree next to the Platelet Aggregation heading click the Add button. The In Vitro Characterization-Platelet Aggregation page opens.
The following table lists and describes the fields on the form.
Some fields may contain the Other option. Click Other to enter an alternative to the selections listed. At that point,your entry is sandwiched by [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear

Characterization Source
Enter the source from which the characterization data is derived, such as a vendor or a laboratory (NCL).

Enter a unique and meaningful name for the characterization. The title you enter will display on the Navigation Tree. The asterisk indicates a required field.

Enter the protocol from which the data is derived and or enter identifying version information for the protocol from which the data is derived. A hyperlink of the protocol text file may appear. Click the hyperlink to open or save the file.

Characterization File/Derived Data Information
Click the Add File/Derived Data link to extend the page where you can add characterization files consisting of graphs, spreadsheet or images, or derived data.
Note:
You can add as many files and derived data as you wish, or you can add derived data without adding files.
File Type
Specify the type of characterization file.
Data Category
These options are dependent upon the characterization type. For example, for Size, options include Intensity Distribution, Number Distribution and Volume Distribution. Select the choice appropriate for your nanoparticle.
File Name
This field remains empty until you select the file to be uploaded. To do so, click the Load File button, opening the Load Characterization File page. For more information about completing the form, see Loading Physical Characterization Files.
File/Derived Data Description
Enter an appropriate description for the file or derived data.
Add Derived Data
Click the hypertext link to open fields for selecting or entering names or values for the derived data. These fields can be specific for the type of characteristic being defined. For example, the Name drop down list is appropriate for Size characterizations: PDI, Peak I, RMS size and Z Average.
Copy to Other Particles
If appropriate, this option copies files and summary information to one or more selected particles from the same vendor. Select one or more particle(s) to which the copied data is to be transferred. (Only appropriate particles are listed.) For more information, see Copying Characterizations.
Also Copy Derived Data?
Select this button to copy derived data, as well, to the selected particle.
In the Navigation Tree, a hypertext link for the new characterization is added under Platelet Aggregation.
