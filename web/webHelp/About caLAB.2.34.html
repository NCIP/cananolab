Downloading Files
Once a file has been uploaded into caNanoLab, it can be downloaded.
To download a file, follow these steps:
1.
In the Run Tree, locate the run that has the file(s) you want to download. Click the (+) symbol in front of the run, expanding the folder to show the In and Out folders.
2.
Open the folders to identify the file(s) to be downloaded.
3.
Click the Download Files hypertext link in the right pane of the browser. The browser displays all uploaded files in the folder.
4.
Click the Download link corresponding to the file to be downloaded. (Or click Download All Files to download all files in the current folder).
5.
Navigate to the location where your downloaded file(s) will be saved.
If you select the option to download all files, your downloaded files will be zipped.
