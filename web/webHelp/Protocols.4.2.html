Protocols Overview
If you do not have certain security privileges, the Creating a New Sample link does not appear on the Manage Sample/Aliquote Home page.
The Protocols function enables you to create or search for a protocol. A protocol provides clear and detailed documentation about the precise actions taken in creating or annotating a nanoparticle. When you click the Protocols option on the caNanoLab menu bar, the Manage Protocol Home page appears. The Protocol Links on the Manage Protocol Home page provide the following options:
•
Creating a New Protocol: This takes you to a form for entering information for a new protocol you want to submit. See Submitting a Protocol.
•
Search Existing Protocols: This takes you to a form for launching a search for an existing protocol. For more information, see Searching Existing Aliquots.
For more information about protocols, see Description of a Protocol.
