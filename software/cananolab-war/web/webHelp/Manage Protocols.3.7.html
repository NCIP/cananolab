Protocol Update Page
The Protocol Update page opens when you click a Protocol Name URI in Protocol Search Results. The page displays all of the information for a protocol that was entered when the protocol was created in caNanoLab (Submitting a Protocol). With appropriate permissions, you can edit the title, description and visibility of a protocol in this dialog box.
To open an associated protocol file, click the hypertext file name in the Protocol File field. An intermediate dialog box may open, requesting that you select the application with which to view the file.
