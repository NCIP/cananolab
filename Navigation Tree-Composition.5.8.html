Composition File
This section of the Sample Composition sidebar options allows you to add files relevant to the nanoparticle sample independently of the entity or chemical associations that you have defined. You can add as many files as you wish. To add a file, follow these steps. (Asterisks indicate required fields.)
1.
Click the Composition File button in the Sample Composition section of the left sidebar.
2.
Fill in each of the following fields on the Composition File Information form:
Composition File Fields
Definition
Upload File
or
Enter URL
Click the Browse button to navigate to the file to be associated with the report, or enter the URL where the report is located.
File Type*
Select the file category from the drop-down menu.
File Title*
Enter the file title.
Keywords
Keyword(s) you identify to associate with the file.
Visibility
Select the group(s) to be granted access to the selected file.
