Browsing Public Information
From the Welcome to caNanoLab Login Page, you can browse publicly available nanoparticle protocol, characterization data, and report results. To browse public data, follow these steps:
Once you enter the public caNanoLab area, the menu bar enables you to navigate between the Protocols, Nanoparticles, and Reports functions. Click Home to return to the Welcome to caNanoLab Login Page.
To browse public nanotechnical protocols, click the arrow for further instructions
1.
Click the Protocols (Search) link.
2.
Fill in the Search Protocols page (see Searching Existing Protocols) and click the Search button.
The Protocol Search Results page appears (see Protocol Search Results).
3.
To display information about the protocol, click the Protocol Name link.
The Protocol File page appears (see Protocol Update Page).
4.
To open the associated file, click the hypertext file name in the Protocol File field. An intermediate dialog box may open, requesting that you select the application with which to view the file.
To browse public nanoparticle characterization data, click the arrow for further instructions
1.
Click the Nanoparticles (Search) link.
2.
Fill in the Search Nanoparticles page (see Searching Existing Nanoparticles) and click the Search button. To perform a search of remote hosts, see Performing a Remote Nanoparticle Search.
The Nanoparticles Search Results page appears (see Nanoparticle Search Results).
3.
To display information about the protocol, click the Protocol ID link.
The General Information page appears (see Submitting a Nanoparticle Sample).
4.
To open the associated file, click the hypertext file name in the Protocol File field. An intermediate dialog box may open, requesting that you select the application with which to view the file.
To browse public documents, click the arrow for further instructions
Documents can be either publications or reports that detail results of nanoparticle characterization projects.
1.
Click the Documents (Search) link.
2.
Fill in the Search Documents page (see Searching Nanoparticle Documents) and click the Search button.
The Nanoparticle Document Search Results page opens (see Nanoparticle Documents Search Results).
3.
To display information about the document, click the Document Title link.
The Report or Publication File page appears (see Document Update Page).
4.
To open the associated file, click the hypertext file name in the Publication Link or Report Link field. An intermediate dialog box may open, requesting that you select the application with which to view the file.
